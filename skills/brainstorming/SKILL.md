# Brainstorming

1. **Diverge** - generate several distinct approaches, not variations of one.
2. **Compare** - for each, name what it optimizes and what it trades off.
3. **Converge** - recommend one, with the reason, and note what would change the choice.

Surface options to the user before locking a direction. Avoid anchoring on the first idea.
